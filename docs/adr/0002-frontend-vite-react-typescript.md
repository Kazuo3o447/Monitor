# 0002 Frontend with Vite, React and TypeScript

## Status
Accepted

## Kontext
Das aktuelle Frontend ist ein statisches Vanilla-JS-Setup. Fuer weitere Views, Routing und testbare Komponenten ist ein strukturiertes Frontend-Framework erforderlich.

## Entscheidung
Ab Phase 1 wird das Frontend auf Vite + React + TypeScript umgestellt. Tailwind wird als Build-Dependency genutzt.

## Konsequenzen
- Bessere Komponentisierung und Wartbarkeit.
- Schnellere lokale Entwicklung mit modernem Tooling.
- Klare Trennung zwischen UI-State und Server-State.

## Alternativen
- Vanilla-JS weiterfuehren: verworfen wegen steigender Komplexitaet.
- Andere Frameworks: aktuell nicht priorisiert.
